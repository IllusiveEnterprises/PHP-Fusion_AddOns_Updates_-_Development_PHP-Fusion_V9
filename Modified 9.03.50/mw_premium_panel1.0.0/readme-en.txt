---------------------------------------
1. Installation
---------------------------------------

The folder "infusions" must be in the
"root" directory of the PHP-Fusion installation can be uploaded.

System Admin -> Infusions -> mw_premium_panel installation.

Now you can make the desired settings in the admin under Infusions -> MW Premium.
