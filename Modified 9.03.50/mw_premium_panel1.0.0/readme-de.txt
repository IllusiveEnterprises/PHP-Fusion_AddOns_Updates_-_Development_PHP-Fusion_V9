Ein Premium system für PHP-Fusion v7.02.xx

---------------------------------------
1. Installation
---------------------------------------

Der Ordner "infusions" muss in das
"root"-Verzeichnis der PHP-Fusion Installation hochgeladen werden.

System Admin -> Infusionen -> mw_premium_panel installieren

Nun könnt Ihr noch im Admin unter Infusionen -> MW Premium die gewünschten einstellungen vornehmen.

Viel Spaß!
