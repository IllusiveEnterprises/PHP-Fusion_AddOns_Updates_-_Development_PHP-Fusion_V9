Für PHP Fusion Version 7.x
by DeeoNe (http://www.deeone.de)
--------------------------------------

-------------
0. Einleitung
-------------

Ein komplettes Premium-System für PHP-Fusion v7.xx.xx
---------------------------------------
1. Features
---------------------------------------
Mitglied:
- Mitglieder können mit ihren Scores Premiumzeiten aktivieren.
- Mitglieder werden voll automatisch in einer Benutzergruppe hinzugefügt.
- Nach ablauf des Premiums werden die Mitglieder wieder aus der Gruppe automatisch entfernt.
- Benachrichtigung bei Scores kauf.
- Überarbeitetes Panel mit Grafik möglich.
- Anzeigen von Vorteilen die der Admin eingetragen hat.

Admin:
- Einstellungen der Preise und dauer der einzelnen Pakete
- Usergruppe in der die Mitglieder kommen kann eingestellt werden.
- Benachrichtigung bei Scores kauf von Mitgliedern.
- Mitglieder Aktivieren, Deaktivieren und löschen.
- Mitglieder Premium Zeit + -
- Mitglieder Premium auf Zeit schenken.
- Eintragen von Premium Vorteilen.
- Premium Grafik ein und aus schalten.
- Anzeigen von Bestätigungen verbessert.
---------------------------------------
2. Installation
---------------------------------------

Die komplette Ordnerstruktur muss in das
"root"-Verzeichnis der PHP-Fusion Installation kopiert
werden.

Danach kann die Infusion über System Admin -> Infusionen
installiert werden.

Nach der Installation muss dann nurnoch das Panel angelegt werden und schon kann es losgehen.

OPTIONAL Premium Profilfeld aktivieren: User Admin -> Profilfelder -> Premium Status: aktivieren 
	
---------------------------------------
3. Hinweise
---------------------------------------

Diese Infusion bzw. das Panel ist vorrübergehend erstellt worden, da es mit der anderen Premium-version noch etwas dauern kann.
