Scores time bonus for PHP-Fusion v9.03.50

---------------------------------------
1. Installation
---------------------------------------

The complete folder structure (infusions) must be copied
in the "root" directory of the PHP-Fusion installation. (in case of update overwrite all files).

System Admin -> Infusions-> D1 Time Panel install

The panel is created automatically.

Then you open the admin of the infusion: Infusions -> D1 Time System,
there you have to register this infusion (Infusions Activator).

If the registration and activation were successful, the admin menu of the infusion opens and the functions of the infusion are now activated.

Then you can make the desired settings and then
Check whether the panel is at the desired location.


Ready! ENJOY

2. Update
---------------------------------------

Update 1.0.0 -> 1.1.0: Upload (overwrite) all files and install as an update under System Admin -> Infusions -> D1 Time Panel.
