0. Einleitung

Ein Coupon System für PHP-Fusion v7.01.xx & v7.02.xx
---------------------------------------
1. Features
---------------------------------------

In Planung ein Gutschein System, wo man per Gutscheincode Scores & Premium Zeit bekommt.

---------------------------------------
2. Installation
---------------------------------------

Die komplette Ordnerstruktur (administration, infusions) muss in das
"root"-Verzeichnis der PHP-Fusion Installation kopiert
werden.

Danach kann die Infusion (D1 Coupon Panel) über System Admin -> Infusionen
installiert werden.

Das Panel wird Automatisch ertsellt.

Danach Öffnet ihr den Admin der Infusion: Infusions -> D1 Coupon System,
dort müsst ihr dann diese Infusion Registrieren (Infusions Activator).
Wenn die Registrierung und die Aktivierung erfolgreich war, öffnet sich das Adminmenü der Infusion und die Funktionen der Infusion sind nun aktiviert.
