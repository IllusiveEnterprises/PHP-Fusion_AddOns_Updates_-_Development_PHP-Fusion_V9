-------------
0. Einleitung
-------------

Ein Panel um Scores für das Scoresystem zu kaufen für PHP-Fusion v7.xx.xx
Über: Bankdaten, PayPal, micropayment.

---------------------------------------
1. Installation
---------------------------------------

Der Ordner "infusions & administration" muss in das
"root"-Verzeichnis der PHP-Fusion Installation hochgeladen werden (beim Update alle Dateien überschreiben).

System Admin -> Infusionen -> mw_buy_points_panel installieren

Das Panel wird automatisch erstellt.

Nun könnt Ihr noch im Admin unter Infusionen -> MW Buy Points die gewünschten einstellungen vornehmen und dann
prüfen ob das Panel an der gewünschte stelle ist.

Viel Spaß!
