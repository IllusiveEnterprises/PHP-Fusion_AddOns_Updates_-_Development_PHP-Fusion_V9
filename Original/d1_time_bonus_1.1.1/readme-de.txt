0. Einleitung
-------------

Scores Zeit Bonus für PHP-Fusion v7.xx.xx

---------------------------------------
1. Installation
---------------------------------------

Der Ordner "infusions & administration" muss in das
"root"-Verzeichnis der PHP-Fusion Installation hochgeladen werden (beim Update alle Dateien überschreiben).

System Admin -> Infusionen -> D1 Time Panel installieren

Das Panel wird automatisch erstellt.

Nun müsst Ihr noch im Admin unter Infusionen -> D1 Time Panel diese Infusion Registrieren!

Danach könnt ihr die gewünschten einstellungen vornehmen und dann noch
prüfen ob das Panel an der gewünschte stelle ist.


Viel Spaß!

2. Update
---------------------------------------

Update 1.0.0 -> 1.1.0: alle Dateien hochladen (überschreiben) und unter System Admin -> Infusionen -> D1 Time Panel als Update Installieren.
Update 1.1.0 -> 1.1.1: Alle Dateien hochladen (überschreiben) und als Update installieren.
