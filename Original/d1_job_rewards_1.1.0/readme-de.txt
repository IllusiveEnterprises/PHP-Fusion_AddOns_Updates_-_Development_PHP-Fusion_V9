Job Verscorung für PHP-Fusion v7.xx.xx

---------------------------------------
1. Installation
---------------------------------------

Der Ordner "infusions & administration" muss in das
"root"-Verzeichnis der PHP-Fusion Installation hochgeladen werden.

System Admin -> Infusionen -> D1_job_rewards_panel installieren

Das Panel wird automatisch erstellt.

Zum schluss den Admin dieser Infusion öffnen und die Infusion Registrieren!.

Nun könnt Ihr noch im Admin unter Infusionen -> D1 Job Rewards die gewünschten einstellungen vornehmen und dann
prüfen ob das Panel an der gewünschte stelle ist.

Viel Spaß!
