0. Einleitung
-------------

Eine Pinwannd für PHP-Fusion v7.xx.xx

---------------------------------------
1. Installation
---------------------------------------

Der Ordner "infusions & administration" muss in das
"root"-Verzeichnis der PHP-Fusion Installation hochgeladen werden.

System Admin -> Infusionen -> D1_pinnwand_panel installieren

Das Panel wird automatisch erstellt.

Nun müsst Ihr noch im Admin unter Infusionen -> D1 Pinnwand die Infusion Registrieren.
Nach erfolgreicher Registration könnt ihr die gewünschten einstellungen vornehmen und die Infusion ist nun voll Funktionsfähig.


Viel Spaß!

---------------------------------------
2. Hinweis
---------------------------------------
Diese Pinnwand beinhaltet eine Addon Option für das SuN Voter Addon 2.0 das hier zu bekommen ist:
http://www.phpfusion-support.de/infusions/fusion_downloads/dl_details.php?dl_id=298
Die Pinnwand erkennt automatisch ob das Addon installiert ist und schaltet sich automatisch ein.

Diese Infusion hat ein Importer im Admin, um die alten Posts vom TI Blog zu Importieren.
Im Infusionsordner ist ein Ordner Namens ti_blog_system dieser ist dazu da um ein Fix aufzuspielen,
das beim deinstallieren des TI Blogs die Kommentare nicht gelöscht werden, die beim Importieren gebraucht werden.
Ist dies eine Neuinstallation OHNE Import nutzung, kann man den Ordner ti_blog_system auch löschen.

ACHTUNG: Erst müssen die TI Blogs Importiert werden, dan darf erst das TI Blog System deinstalliert werden, sonst sind alle vorhandene Blogs weg.
